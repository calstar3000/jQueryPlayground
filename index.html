Let's have some fun with jQuery

.addClass(className)

Description

Add a class to an element.

Code

$("#btnAddClass").click(function() {
    $(this).addClass("pure-button-primary");
});

Demo


.attr(attributeName)

Description

Get an element's attribute value for a given attribute name.

Code

$("#btnAttr").click(function() {
    $("#txtAttr").html($(this).attr("class"));
});

.fadeTo(duraction, opacity, easing, complete)

Description

Adjust the opacity of an element.

Code

    $("#pnlFadeTo").fadeTo(
        1000, // Length of animation
        0.2, // Opacity to fade to
        "linear", // Easing function for animation
        function() {
            console.log("fadeTo() finished!");
        } // Function to call once animation is complete
